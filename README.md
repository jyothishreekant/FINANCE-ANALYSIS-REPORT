# 💰 Finance Analysis Dashboard | Power BI

A professional **Finance Analysis Dashboard** built using **Power BI** to analyze financial transactions, customer behavior, revenue, fees, taxes, and transaction performance. This interactive dashboard provides meaningful business insights through KPIs, charts, and filters that help stakeholders make data-driven decisions.

---

## 📌 Project Overview

The Finance Analysis Dashboard is designed to monitor and analyze financial transactions across different customer segments, states, genders, and transaction types.

The dashboard enables users to:

- Track Total Revenue and Transaction Performance
- Analyze Fees and Taxes
- Monitor Monthly Financial Trends
- Compare Customer Segments
- Evaluate Transaction Status
- Explore Detailed Transaction Records

## 📈 Key KPIs

| KPI | Description |
|------|-------------|
| Total Amount | Overall transaction amount |
| Total Transactions | Number of completed transactions |
| Average Transaction Value | Average value per transaction |
| Total Fee | Total fees collected |
| Total Tax | Total tax collected |
